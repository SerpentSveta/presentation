CSS Grid Layout

I want to show my presentation about CSS Grid Layout.

CSS Grid Layout widely available across major browsers.

CSS grid layout introduces a two-dimensional grid system to CSS.

Benefits of CSS grid layout

  • High degree of control over the location and size of the elements
  • Flexible and adaptive layout
  • Clear and short code

display: grid

A grid is created on a container that contains grid elements.

grid-template-columns

Columns in a grid are created using the grid-template-columns property.

repeat()

If you need identical columns or rows, you can use the repeat() function.

grid-template-rows

Rows in a grid are created using the grid-template-rows property.

gap

The distance between cells can be set using the gap property.

the «fr» unit

We can use px together with fr.

grid-auto-flow

Row is by default.

grid-auto-rows, grid-auto-columns

The grid-auto-rows CSS and grid-auto-columns properties specifies the size of an implicitly-created grid row or column track or pattern of tracks.

grid-auto-rows, grid-auto-columns

auto-fill, auto-fit

The CSS properties auto-fill and auto-fit allow you to create a grid with as many tracks of a given size as will fit in the container.

grid-column-start, grid-column-end,
grid-row-start, grid-row-end

The grid-column-start CSS property specifies a grid item's start position within the grid column by contributing a line.

grid-column, grid-row

The grid-column CSS shorthand property specifies a grid item's size and location within a grid column, thereby specifying the inline-start and inline-end edge of its grid area.

justify-items: stretch

The stretch value is used by default.

justify-items: center

justify-items: start

justify-items: end

align-items: stretch

The stretch value is used by default.

align-items: center

align-items: start

align-items: end

grid-areas

A grid area is one or more grid cells that make up a rectangular area on the grid.
