CSS Grid Layout

CSS grid layout introduces a two-dimensional grid system to CSS. Grids can be used to lay out major page areas or small user interface elements.

Benefits of CSS grid layout

  • High degree of control over the location and size of the elements
  • Flexible and adaptive layout
  • Clear and short code
